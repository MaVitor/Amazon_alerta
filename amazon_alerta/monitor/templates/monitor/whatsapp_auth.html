Autenticação do WhatsApp

Como autenticar o WhatsApp para envio de notificações

Este sistema usa o WhatsApp Web para enviar notificações de preços. Para autenticar, siga os passos abaixo:

1. Inicie o processo de envio de notificação

Para iniciar o processo de autenticação, você precisa tentar enviar uma notificação de teste ou executar o comando de verificação de preços:

  • Envie uma notificação de teste para um contato através da interface do sistema
  • OU execute o comando python manage.py check_prices no terminal

2. Escaneie o QR Code

Quando você iniciar o processo, uma janela do navegador será aberta automaticamente com o WhatsApp Web. Você verá um QR code na tela.

  • Abra o WhatsApp no seu celular
  • Toque em Menu (três pontos) ou Configurações
  • Selecione "Dispositivos conectados"
  • Toque em "Conectar um dispositivo"
  • Aponte a câmera do seu celular para o QR code exibido na tela do computador

Nota importante: Você só precisa escanear o QR code uma vez. O WhatsApp Web manterá a sessão ativa por um período, geralmente de algumas semanas, a menos que você desconecte manualmente.

3. Pronto!

Após escanear o QR code com sucesso, o sistema estará autenticado e poderá enviar notificações via WhatsApp.

Solução de problemas

Se você encontrar problemas durante a autenticação:

  • Certifique-se de que seu celular tem conexão com a internet
  • Verifique se o WhatsApp no seu celular está atualizado
  • Tente fechar e reabrir o WhatsApp no seu celular
  • Se a janela do navegador não abrir automaticamente, tente executar o comando novamente
  • Se o problema persistir, tente reiniciar o servidor Django e tentar novamente
